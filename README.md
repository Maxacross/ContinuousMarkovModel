# Симулятор безперервних марковських ланцюгів

Цей веб-застосунок дозволяє моделювати **безперервні марковські ланцюги** на основі матриці інтенсивностей та вектора початкових ймовірностей. Можна візуалізувати граф ланцюга, обчислювати стаціонарні ймовірності та будувати графіки динаміки ймовірностей у часі.

## Можливості

- Редагування матриці інтенсивностей переходів
- Задання вектора початкових ймовірностей
- Візуалізація графа марковського ланцюга (SVG / PNG)
- Формування системи рівнянь Колмогорова
- Розв'язок системи та обчислення стаціонарних ймовірностей
- Побудова графіків зміни ймовірностей станів у часі
- Збереження та завантаження моделей у форматі JSON

## Використання

1. Відкрийте `index.html` у веб-браузері.
2. Встановіть кількість станів, кроків та кінцевий час моделювання.
3. Введіть матрицю інтенсивностей та вектор початкових ймовірностей.
4. Натисніть кнопку **"Обчислити"**, щоб отримати результати та візуалізацію.
5. Завантажте граф або збережіть модель для подальшого використання.

## Залежності

- [MathJax](https://www.mathjax.org/) – для рендерингу LaTeX формул
- [Math.js](https://mathjs.org/) – для обчислень з матрицями
- [Viz.js](https://viz-js.com/) – для візуалізації графів
- [Chart.js](https://www.chartjs.org/) – для побудови графіків ймовірностей
